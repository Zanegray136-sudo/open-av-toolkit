# Open AV Toolkit

Open AV Toolkit is a browser-based LED engineering and production toolkit built with React and TypeScript.

## Features

* LED screen configuration
* Resolution calculations
* Power and weight calculations
* Processor analysis
* Data routing maps
* Snake numbering
* Patch sheet generation
* Project management tools
* Engineering report generation

## Roadmap

* Save / Load Projects
* Venue Numbering (A01, B01, C01)
* Multi-Processor Support
* Advanced PDF Reports
* Routing Export Tools
